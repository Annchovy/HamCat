Dataset view shows what dataset is being displayed and the slider for the inclusion of missing data. Move the slider to adjust the missingness threshold and include data items with missing levels below or equal to the set threshold.
Attribute view provides an overview of the attributes in your dataset along with their levels and level counts.
Item view includes two parts: an ego Hamming network and an entropy map. The ego network allows you to explore your data items grouped based on their attribute levels. Data items with identical levels across all attributes are grouped together into a node.
We view the grouped data items – the nodes - from the “perspective” of the focal node located in layer 0. Initially, the most frequent data item is chosen as the focal node. However, you can select a different node of the network as the focal node by clicking on it. The rest of the nodes is placed across the layers based on their dissimilarity to the focal node – the number of the differing attribute levels. The more dissimilar nodes to the focal node the farther they are from it in the graph.
